Welcome to Pearls! Quick Start Guide

Installation

You can install the script in seconds by using our professional Pearls! installer found at 

http://www.your-domain.com/pearls/install/
first, it will make sure that your server meets the required prerequisites for runnig pearls, then it will guide you through 4 simple steps and the script will be installed automatically, make sure you don't re-load the page in any step.
after installation, you have to delete (install) folder to avoid any security problems

Admin Settings

After you install the script, login through 

http://www.your-domain.com/pearls/login/

Then access admin dashboard through 
http://www.your-domain.com/pearls/admin/

There you'll find many options that gives you control over every section of the script.

General Settings

  • Site Name Let's you rename the whole app for branding
  • Site description Describe your app in many words, useful for SEO
  • Site Status You may close the whole site and make it visible only for admins through this option
  • Closure Message Your custom message that will appear to visitors and registered users if you close the site
  • URL Type Choose wheather you want topics called via their Slug (default, very useful for SEO), or via their ID
  • Publishing a question Gives you the ability to make users publish question immediately or after moderation by admins
  • Publishing an answer Gives you the ability to make users publish answers immediately or after moderation by admins
  • Public access without login Enable it if you want unregistered visitors to see the questions and answers without need for registration, yet they will view only 1 answers for every question
  • Default registration group Defines what privileges registered user will get, you can change rules and privileges through our smart privileges system that will be discussed in a moment

Pending Approval

This page contains questions and answers that still not approved by admins, needles to say that it's only active if you chose to moderate questions and answers before publishing as explained eariler

Users section

You can view your registered users there, also you can edit their profiles.
Also, through this screen, you can Change User Privilege Groups (to promote users to admins for example!) just click edit and change privilege group to desired level

Privileges section

This section gives you complete control over who can see what!
You can edit what a registered user can see and what a guest can access .. You can also create new privilege groups and assign users to them like a limited admins or special members, you can control evey single function of the application through this engine

Users Registration

Users can register new accounts through login screen, simply press on (Register a new account)
Every account must have a unique Email, registration script uses CAPTCHA to fight against bots and auto-submission scripts, so you must set Google CAPTCHA keys properly during installation.

Posting Questions

You can post questions through typing them in search form, script automatically searches questions database and if no questions relative to the words entered you can post your questions directly by clicking (Ask!)
Also there's a big input box in your homepage for directly asking questions to the world
(Slug) is the part that would appear in the URL of the question, useful for SEO.

Likes/Dislikes and Follows

One of the coolest functions in pearls, is that users can interact with it throug like/dislike questions, answers and follow feeds/users to get notified for any new questions or answers posted.
